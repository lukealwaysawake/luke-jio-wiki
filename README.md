# 지오와 동현 - 관계위키

나무위키식 문서 감성으로 정리한 Luke/Jio 관계 위키 페이지.

## 반영된 핵심 날짜

- 2026-05-19: 구리 심야 방문
- 2026-05-21: 청담 오니바, 꽃, 케이크, 고백, 공식 연애 시작
- 2026-05-22: 점심과 수족관
- 2026-06-23: 청계천 인근 라운지 사진/영상, 제로컴플렉스 디너 코스, 선물, 영쉘던, 아사이볼, 복귀
- 2026-06-27~28: 북촌 한식 오마카세, 기가스, 한옥호텔, 파스타, 카페, 사진 자료 10장

## 기능

- 문서 상단 나무위키식 접이식 목차
- 데이트 기록 내부 접이식 날짜 목차
- 데이트 기록별 브라우저 로컬 댓글

## Media note

2026-06-23 및 2026-06-27 사진·영상 자료는 원본 분위기를 유지하는 방향으로 반영했다.
